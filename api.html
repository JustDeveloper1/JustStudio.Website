JustStudio.AdminPanel

Adding admin panel to your game

Create a server script in ServerScriptService and follow the instructions below, or add this model to your game.
Note: The game owner always has access to the admin panel (even without configuring the admin panel).
require ( 18408482320 ). Initialize ()

Adding Settings (Optional)

You can add admins by roles in your group. And you can toggle admin anchor when admin panel open.
Note: Adding group settings is optional.
require ( 18408482320 ). Initialize ({
      -- Settings
      [ "Admin" ] = {
            [ "AnchorPlayer" ] = true ; -- Anchor admin when admin panel open? (Default: true)
      };
      [ "Group" ] = {
            [ "GroupID" ] = 34450183 ; -- Your Group ID
            [ "AdminRoleNames" ] = {
                  "ModTeam" ; "DevTeam" ; "Owner" ; -- Up to 3 roles (USE ROLE NAMES ONLY!)
            };
      };
})

Adding Admin IDs (Optional)

You can add admins by UserIDs.
Note: There should be settings first, and only after settings you can add Admin IDs!
require ( 18408482320 ). Initialize ({
      -- Settings
      [ "Admin" ] = {
            [ "AnchorPlayer" ] = true ; -- Anchor admin when admin panel open? (Default: true)
      };
      [ "Group" ] = {
